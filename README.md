## Installation and Setup 

1. First, ensure Playwright is installed:
```bash
pip3 install playwright
playwright install
```
2. Install pytest and pytest-playwright
```bash
pip3 install pytest pytest-playwright
```
3. Run with pytest
```bash
pytest test_login.py
```
